# Animated Portfolio Website

A modern and fully responsive animated portfolio website developed for CODTECH Web Development Internship Task-1.

## 🚀 Features

- Modern UI Design
- GSAP Smooth Animations
- Responsive Layout
- Glassmorphism Effects
- Animated Skill Cards
- Project Showcase Section
- Contact Form
- Particle Background Animation
- Floating Cartoon Elements
- Social Media Icons

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript
- GSAP
- Particles.js
- Font Awesome
- Google Fonts

## 🎯 Purpose

This project was created as part of the CODTECH Web Development Internship to demonstrate frontend development skills, responsive design, and advanced animations.
